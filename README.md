# 🧪 Slime Combat

**Slime Combat** is a fast-paced **2-player arena fighting game** where two slimes battle it out for victory!  
Built using **GameMaker**, this project focuses on player movement, collision handling, and responsive combat mechanics.

---

## 🎮 Gameplay Overview

Two slimes face off in a small arena — jump, dash, and attack your opponent to drain their health.  
The first slime to reach zero health loses.  
It’s simple, fun, and chaotic!

---

## ✨ Features

- 🕹️ **Local 2-player combat**  
- 💥 **Attack & defense system** with hit detection  
- ⚡ **Power-ups** that spawn randomly in the arena  
- 🎨 **Custom sprites** and simple particle effects  
- 🔊 **Sound effects** for hits, jumps, and victories  
- 🧭 **Health bars** and round restarts  

---

## 🎯 Controls

### Player 1  
| Action | Key |
|--------|-----|
| Move Left | **A** |
| Move Right | **D** |
| Jump | **W** |
| Attack | **F** |

### Player 2  
| Action | Key |
|--------|-----|
| Move Left | **←** |
| Move Right | **→** |
| Jump | **↑** |
| Attack | **Enter** |
